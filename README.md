# WebSocket 連線測試工具

🔌 WebSocket Test Tool - 快速測試 WS/WSS 連線

## 🌐 網站網址
https://lance70176.github.io/websocket-test-tool/

## ✨ 功能特色

- 🔗 **雙協議支援** - WS (未加密) / WSS (加密)
- ⚡ **即時連線** - 顯示連線時間與狀態
- 📊 **連線資訊** - 協議、狀態、耗時一目了然
- 📝 **詳細日誌** - 時間戳記 + 圖示化顯示
- 🎯 **自動測試** - 連線後自動發送 ping 測試
- ⏱️ **自動關閉** - 10 秒後自動斷開連線

## 🚀 使用方法

1. 輸入 WebSocket 伺服器網址（如：`gate348-mg.fpfyinvest.com`）
2. 點擊「測試 WS 連線」或「測試 WSS 連線」
3. 查看連線狀態與日誌

## 📋 連線狀態說明

| 狀態 | 說明 |
|------|------|
| 連線中 | 正在嘗試建立連線 |
| 已連線 | 成功建立 WebSocket 連線 |
| 已關閉 | 正常關閉連線 |
| 異常關閉 | 連線異常中斷 |
| 錯誤 | 發生連線錯誤 |

## 🛡️ 隱私說明

- 純前端測試，所有連線由瀏覽器直接發送
- 不會傳送到任何第三方伺服器
